## Clone the Repository:

Navigate to master branch of project
Copy the repository URL ([URL of Repo](https://github.com/Umar888/kanban-board.git)).
Open your terminal or command prompt.
Use the git clone command followed by the repository URL to clone the repository locally:

git clone https://github.com/Umar888/kanban-board.git

## Navigate to the Project Directory:

Once cloned, navigate into the project directory using the cd command:

cd your-repository

This command moves your terminal's current location into the cloned repository directory.

## Install Dependencies:

Make sure flutter SDK is installed on your system. You can download flutter sdk from [here](https://docs.flutter.dev/get-started/install?gad_source=1&gclid=CjwKCAjwmrqzBhAoEiwAXVpgotPqdCbxxnYFDv2aMbkoRQggyDQ8t6dCUM-P_NJLEBPLQTo1Mwzx2hoCACgQAvD_BwE&gclsrc=aw.ds)

Make sure latest android studio is installed on your system. You can download android studio from [here](https://developer.android.com/studio?gad_source=1&gclid=CjwKCAjwmrqzBhAoEiwAXVpgojzBVaSTS8_OA2wY9cCTzRXfQ-yfDqWCepl7xzHm1cYVedC1MP1ZaxoCYyoQAvD_BwE&gclsrc=aw.ds)

## Setup Project:

Open Android Studio and go to settings

<img width="291" alt="Screenshot 2024-06-16 at 1 50 32 PM" src="https://github.com/Umar888/kanban-board/assets/44860615/88ca3599-ccd3-4efd-95b2-31b6208f3e47">


In Settings, go to Languages and Frameworks -> Flutter

<img width="970" alt="Screenshot 2024-06-16 at 1 51 42 PM" src="https://github.com/Umar888/kanban-board/assets/44860615/c2410f0a-6357-4f7f-97ae-4b9a83ef309a">


In Flutter, Click on Flutter SDK Path and add path of downloaded flutter SDK and Click OK

<img width="730" alt="Screenshot 2024-06-16 at 1 52 48 PM" src="https://github.com/Umar888/kanban-board/assets/44860615/5c1476e5-39de-4339-9752-ea141b4892c2">


Then click on tools -> Flutter -> Flutter Clean

<img width="547" alt="Screenshot 2024-06-16 at 1 53 22 PM" src="https://github.com/Umar888/kanban-board/assets/44860615/138e269f-2986-46fd-bbe6-b0051abac007">


After Cleaning complete, Then click on tools -> Flutter -> Flutter Pub Get

After executing complete, open the terminal from Android Studio 

<img width="994" alt="Screenshot 2024-06-16 at 1 55 47 PM" src="https://github.com/Umar888/kanban-board/assets/44860615/655f91a5-5a81-478b-aad3-cf996f0e94aa">


And Run following command

flutter pub run build_runner build

After completing exection, run following command

flutter gen-l10n

After executing the commands, connect yous system with physical device or open simulator(If you are using macOS) or download emulator(If you are using windows)

Once the device connected successfully, then click on play button at top
